## Data Structures

### Binary Tree
A binary tree is a tree data structure where each node has at most two children (left and right).

**Features:**
- Insert, search operations
- Multiple traversals (inorder, preorder, postorder, level-order)
- Height and size calculations

**Location:** `BinaryTree/`

**Usage:**
```bash
cd BinaryTree
python example.py
python test_binary_tree.py
```
